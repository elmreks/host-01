# Host McHostface

A small comparison site for hosting platforms and combo stacks for modern apps, PWAs, auth, payments, databases, and storage.

## Stack

- Vite
- React
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publish

This project is configured for GitHub Pages deployment with:

- repo: `host-01`
- Pages base path: `/host-01/`

A GitHub Actions workflow is included at `.github/workflows/deploy.yml`.

- `data/options.json`
- `content/methodology.md`
